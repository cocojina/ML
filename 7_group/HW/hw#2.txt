기계학습 과제 #2

결정트리 실제 사용 사례: 소셜 미디어 및 광고

사례: 소셜 미디어 플랫폼에서는 결정트리 및 앙상블 학습을 활용하여 사용자의 행동 패턴을 분석하고, 개인화된 광고를 제공하는 데 사용됩니다.

- 데이터 수집: 소셜 미디어 플랫폼은 사용자의 활동을 기록하고 분석합니다. 이에는 클릭, 좋아요, 공유 등의 행동이 포함됩니다.

- 프로파일링 및 세그멘테이션: 수집된 데이터를 기반으로 사용자를 그룹으로 나누어 프로파일링합니다. 이를 통해 유사한 특성을 가진 사용자들이 같은 그룹으로 묶입니다.

- 특성 선택 및 분석: 결정트리와 앙상블 학습 알고리즘은 이러한 데이터를 기반으로 사용자의 행동 패턴을 분석하고 예측합니다. 어떤 특성이 광고에 어떤 영향을 미칠지를 예측하게 됩니다.

- 개인화된 광고 제공: 각 사용자의 프로파일과 행동 패턴을 고려하여 해당 사용자에게 맞는 광고를 제공합니다. 이를 통해 광고의 효과를 최적화하고 사용자의 관심을 끌어냅니다.

- 효과 분석 및 피드백: 제공된 개인화된 광고의 효과를 모니터링하고 분석하여 전략을 최적화합니다. 사용자의 피드백을 토대로 더욱 정교한 타겟팅이 이루어집니다.

이를 통해 소셜 미디어 플랫폼은 사용자들에게 더욱 관련성 높은 광고를 제공함으로써 광고주와 사용자 모두에게 이점을 제공합니다.
















앙상블 학습과 랜덤 포레스트 실제 사용 사례

캐글 경진대회: 대회 "Titanic: Machine Learning from Disaster"

사례: 캐글 경진대회에서 타이타닉호의 생존자 예측 문제를 다룰 때 많은 참가자들은 앙상블 학습을 사용하여 다양한 모델을 결합하고, 랜덤 포레스트를 주로 활용합니다. 랜덤 포레스트는 다양한 특성을 고려하여 승객의 생존 여부를 예측하는 데 사용됩니다.

< 앙상블 학습 활용 >

- 다양한 모델 조합: 참가자들은 다양한 머신러닝 알고리즘을 사용하여 여러 개의 모델을 학습시켰습니다. 이는 로지스틱 회귀, 결정트리, 랜덤 포레스트, 그라디언트 부스팅 등 다양한 모델을 활용한 것입니다.

- 앙상블 구성: 다양한 모델을 조합하여 앙상블을 구성했습니다. 대부분의 경우, 랜덤 포레스트와 그라디언트 부스팅이 주로 사용되었습니다. 이들을 결합하여 높은 예측 성능을 달성했습니다.

- 다수결 투표 또는 평균화: 앙상블에 포함된 모델들이 각각 예측한 값을 조합하여 최종 예측을 수행했습니다. 이는 다수결 투표나 각 모델의 예측값을 평균화하는 방식으로 이루어집니다.

- 모델 성능 평가: 앙상블 모델의 성능은 검증 데이터나 교차 검증을 통해 평가되었습니다. 정확도, 민감도, 특이도 등의 성능 지표를 활용했습니다.

- 모델 튜닝과 최적화: 앙상블에 사용된 개별 모델들의 하이퍼파라미터를 조정하여 최적의 앙상블을 찾아내기 위한 노력이 수행되었습니다.

- 앙상블의 중요성 확인: 모델들의 예측 성능을 개별적으로 평가하고, 앙상블을 사용했을 때의 성능 상승을 확인하여 앙상블의 효과를 입증했습니다.

< 랜덤 포레스트 활용 >

- 특성 선택 및 분석: 랜덤 포레스트를 사용하여 각 특성(성별, 나이, 객실 등급 등)의 중요도를 평가하고, 어떤 특성이 생존 여부를 예측하는 데 중요한 역할을 하는지 확인했습니다.

- 결측치 처리: 랜덤 포레스트를 활용하여 나이 등의 결측치를 예측하는데 활용했습니다. 다른 특성들을 기반으로 결측치를 예측하여 데이터를 보완했습니다.

- 모델 학습: 랜덤 포레스트 알고리즘을 사용하여 다수의 결정트리를 학습시킵니다. 각 트리는 서로 다른 특성의 부분집합을 사용하여 학습되며, 다양한 관점에서 데이터를 분석합니다.

- 앙상블 구성: 랜덤 포레스트를 이용하여 여러 개의 결정트리를 결합하여 생존 여부를 예측합니다. 이를 통해 모델의 예측력을 향상시킵니다.

- 모델 평가: 테스트 데이터에 대한 예측을 수행하고, 정확도, 정밀도, 재현율 등의 성능 지표를 사용하여 모델의 성능을 평가합니다.

- 모델 최적화: 하이퍼파라미터 튜닝 등을 통해 랜덤 포레스트의 성능을 최적화합니다.

- 결과 제출: 최종적으로 모델을 사용하여 테스트 데이터에 대한 예측을 생성하고, 결과를 캐글에 제출합니다.
