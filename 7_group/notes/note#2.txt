2장 – 머신러닝 프로젝트 처음부터 끝까지 
1. 테스트셋, 훈련셋
모델이 학습할 데이터를 훈련 세트(training set), 모델의 성능을 테스트하기 위해 사용할 데이터를 테스트 세트(test set)라고 한다.
김해주 : 데이터세트 : 훈련세트 = 8:2 비율이 아닌, 6:4로 훈련시키면 된다. 훈련데이터가 적으면 과적합 문제가 나올 수 있다. 테스트를 줄이고 데이터를 늘려야 한다.
이진아 : 데이터세트와 훈련세트의 비율을 3:7, 6:4, 8:2 등등 다양한 비율로 학습시켜서 정확도를 높이면 될 것 같다. 
2. 훈련 세트와 데이터 세트를 구분하는 방식 
무작위 샘플링 - 장점 : 다양한 데이터를 학습시킬 수 있다 - 단점 : 정확도가 떨어짐
계층적 샘플링 - 장점 : 정확도가 높아짐 - 단점 : 예외가 발생할 수 있음
